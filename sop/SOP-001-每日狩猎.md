# SOP-001：每日狩猎计划

## 目标
每天自动跟踪AI工具赛道的最新动态，为ToolPilot内容站提供素材。

## XCRN猎场配置

| 猎场 | 关键词 | 频率 |
|:-----|:-------|:----:|
| HackerNews | "AI tool" "new AI" "AI product" | 每4小时 |
| 掘金 | AI工具 推荐 评测 使用体验 | 每4小时 |
| Gitee | AI 工具 开源 新项目 | 每6小时 |
| GitHub Trending | AI tools, LLM applications | 每6小时 |
| arXiv | AI tool, LLM application, prompt engineering | 每12小时 |

## 产出格式

每次狩猎结果存到 `E:\ToolPilot\prey\`，消化后写入日报。

## 日报结构

1. 今日热门AI工具（Top 3）
2. 值得关注的新项目（新冒出来的）
3. 竞品动态（谁在推什么）
4. 关键词热度变化
