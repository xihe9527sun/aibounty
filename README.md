## Phase 3 · 上线部署 ⏳ 下一步

### 3.1 域名方案

| 方案 | 域名 | 年费 | 备注 |
|:-----|:-----|:----:|:-----|
| A | toolpilot.cc | ~¥30 | .cc 便宜，适合技术站 |
| B | toolpilot.cn | ~¥50 | 国内备案友好 |
| C | toolpilot.dev | ~¥80 | .dev 开发者气质 |
| D | toolpilot.ai | ~¥200 | 最切题，略贵 |
| E | toolpilot.tech | ~¥40 | 性价比之选 |

**推荐：** 先用 C（.dev）或 E（.tech）冷启动，验证需求后再升级到 .ai。

### 3.2 托管方案对比

| 平台 | 费用 | 优点 | 缺点 |
|:-----|:----:|:-----|:-----|
| **Vercel** ✅ | **免费** | 自动部署 GitHub，全球 CDN，HTTPS 自动，国内速度一般 |
| Cloudflare Pages | 免费 | 国内速度比 Vercel 好，自带 DDoS 防护 | 需要绑定域名 |
| 腾讯云静态托管 | 免费额度 | 国内访问最快 | 需要备案 |
| GitHub Pages | 免费 | 最简单 | 国内访问不稳定 |

**推荐：** Vercel（主方案，全球访问）+ Cloudflare（国内加速备用）。

### 3.3 部署流程

```
1. 购买域名（Namecheap / 阿里云 / GoDaddy）
       ↓
2. 将源码推送 GitHub 仓库
   └── E:\ToolPilot\ → GitHub → toolpilot/toolpilot
       ↓
3. 关联 Vercel（自动部署）
   └── 每次 push main 自动更新
       ↓
4. 配置自定义域名 + HTTPS
       ↓
5. 配置定时狩猎任务（Windows 任务计划）
       ↓
6. ✅ 上线完成
```

**总耗时：** 域名购买后 30 分钟即可上线。
