🚀 Widget 嵌入

想把 AI 工具推荐放到你的网站上？一行代码搞定：

<!-- 基础：默认 5 个，跟随系统主题 -->
<script src="https://www.aibounty.cn/widget.js"></script>
<!-- 高级：8 个，暗色主题 -->
<script src="https://www.aibounty.cn/widget.js"
  data-count="8"
  data-theme="dark"></script>
⚡ 零外部请求 🎨 浅色/深色/自动 📦 8.8KB 超轻量 🔗 自动外链回 AIbounty

💡 示例：用 fetch 获取数据

// 在浏览器或 Node.js 中获取工具数据
const res = await fetch('https://www.aibounty.cn/data/widget.json');
const data = await res.json();
const topTools = data.tools.slice(0, 5);
topTools.forEach(t => {
  console.log(`${t.t} — ${t.d?.slice(0,30)}`);
});
