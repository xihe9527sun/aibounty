## P0 级：先装上「眼睛」——网站统计

### 1. 百度统计（中国用户首选，免费）

**为什么选它**：中国超过 95% 的搜索引擎流量来自百度，百度统计是国内最主流的分析工具，还能加速百度收录。

**注册方式**（需盘古操作，5分钟）：
1. 访问 https://tongji.baidu.com/
2. 用百度账号登录 → 「新增站点」→ 输入 www.aibounty.cn
3. 获取统计代码（类似 `<script>var _hmt = _hmt || [];...</script>`）
4. 把代码告诉我，我嵌入所有页面

### 2. Google Analytics 4（海外用户，免费）

**为什么选它**：我们有 HN / GitHub / ProductHunt 等海外来源，GA4 可追踪全球流量。

**注册方式**（需盘古操作，5分钟）：
1. 访问 https://analytics.google.com/ → 创建账号 → 创建数据流
2. 获取 Measurement ID（格式 `G-XXXXXXXXXX`）
3. 告诉我，我嵌入所有页面

### 3. Microsoft Clarity（免费热力图）

**可选增强**：Clarity 提供免费的用户点击热力图、滚动深度、会话录制。能看到用户实际怎么用网站。

## P1 级：让搜索引擎找到我们

### 4. 百度自动推送（代码级，我直接做）

百度提供一段 JS 脚本：当用户访问页面时，自动把当前 URL 推送给百度。**零成本、零维护、效果显著**。

✅ **这个我可以直接加到网站，不需要盘古操作**

### 5. 向搜索引擎提交 sitemap

| 搜索引擎 | 提交入口 | 需盘古操作？ |
|---------|---------|:----------:|
| 百度搜索资源平台 | https://ziyuan.baidu.com/ | ✅ 需要登录 |
| Bing Webmaster | https://www.bing.com/webmasters/ | ✅ 需要 Microsoft 账号 |
| Google Search Console | https://search.google.com/search-console | ✅ 需要 Google 账号 |

**建议**：盘古注册上述三个平台后，我在日更管线中增加自动提交 sitemap 步骤。

### 6. JSON-LD 结构化数据（代码级，我直接做）

添加 Schema.org 结构化数据，让搜索结果展示更丰富（星级评分、工具数量、更新时间等）。

✅ **这个我可以直接加**

## P2 级：内容自动分发

### 7. Twitter/X 每日自动发推

每天日报生成后，自动发一条推文，展示今日最佳工具。

**需要**：Twitter API Key / 或盘古给我一个账号的 Cookie/Session

**替代方案**：用 RSS-to-Twitter 机器人（如 IFTTT / Zapier），免费额度够用。

### 8. 掘金专栏同步

我们已经有 掘金投稿 目录。可以每天自动把日报同步到掘金专栏。

**需要**：掘金账号 Cookie（用于文章发布 API）

### 9. 知乎专栏同步

类似掘金，可以每天把精选工具以问答/文章形式同步到知乎。

**建议**：用「如何评价 xxx」或「有哪些好用的 AI 工具」的问答形式，更易获得推荐。

### 10. RSS 生态

我们已经有了 RSS Feed（feed.xml + atom.xml），这是最基础的。可以进一步：

- 提交到 **RSS 阅读器目录**（Feedly、Inoreader 等）
- 提交到 **RSS 聚合导航站**
- 开通 Telegram Channel 自动推送（通过 RSS Bot）

## P3 级：外站链接建设

### 11. 提交到 AI 工具导航站（反向链接）

把自己的网站提交到其他 AI 工具导航站获取外链，如：

| 网站 | 提交链接 |
|------|---------|
| aibot.cn | https://ai-bot.cn/ |
| AI工具集 | https://ai-bot.cn/submit/ |
| 未来百科 | https://nav.6aiq.com/ |
| FuturePedia | https://www.futurepedia.io/ |
| There's An AI | https://theresanaiforthat.com/ |

### 12. 开源社区

在 GitHub 上以「awesome-ai-tools」等形式维护列表，star 积累后自然引流。
