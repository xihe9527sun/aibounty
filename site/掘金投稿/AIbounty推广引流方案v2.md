## 优先级路线图

### P0 — 立即装「眼睛」（需盘古注册，5 分钟）

#### 1️⃣ 百度统计 ⭐ 首选

腾讯云 CDN API 只能看到请求数，看不到用户行为（停留时间、来源渠道、访问深度）。

**百度统计能告诉你**：
- 每天多少人访问、来自哪个城市
- 用户通过什么关键词搜到你的（百度搜索词）
- 哪个页面最受欢迎
- 用户从哪里点进来的（百度搜索 / 直接访问 / 外部链接）

**注册步骤**（盘古做，5 分钟）：
1. 打开 https://tongji.baidu.com
2. 用百度账号登录 → 「新增站点」
3. 输入 `www.aibounty.cn`
4. 复制统计代码给我，我嵌入所有页面
5. 次日就能看到数据

#### 2️⃣ Microsoft Clarity（免费热力图，无需翻墙）

可替代 Google Analytics 的海外方案。
- 免费，无限流量
- 能看到用户点击热力图
- 能看到用户滚动深度
- 可回放用户浏览过程

**注册**：https://clarity.microsoft.com → 添加站点 → 获取代码给我

### P1 — 搜索引擎收录（可部分自动化）

#### 3️⃣ 百度自动推送 ✅ 已上线

昨天已加到所有页面（index/tool/daily/about），新页面自动通知百度。

#### 4️⃣ 百度资源平台（提交 sitemap）

**需盘古注册**（10 分钟）：
1. 打开 https://ziyuan.baidu.com
2. 百度账号登录 → 「站点管理」→ 添加 `www.aibounty.cn`
3. 验证网站所有权（推荐 CNAME 验证，最简单）
4. 验证通过后，把 sitemap 链接告诉我，我自动提交

#### 5️⃣ 必应 Webmaster

Bing 在国内可访问，且 Bing 在 PC 端份额不小。
1. 打开 https://www.bing.com/webmasters
2. Microsoft 账号登录（或 GitHub 账号）
3. 添加站点 → 获取代码 → 我来验证

### P2 — 内容自动分发（我每天帮你发）

#### 6️⃣ 掘金专栏同步 ⭐

掘金是国内最大的中文开发者社区，我们的内容（日报/工具推荐）天然适合掘金受众。

**需盘古**：
1. 注册/登录 https://juejin.cn
2. 开通专栏（设置 → 专栏管理）
3. 把 Cookie 告诉我，我配置自动发布脚本
4. **每天日报生成后自动同步到掘金**，附原文链接

#### 7️⃣ V2EX 自动发布

V2EX 是国内硬核技术社区，发布工具推荐帖引流效果很好。

**需盘古**：注册 https://v2ex.com 账号并告诉我

#### 8️⃣ 知乎专栏

知乎长尾搜索流量很可观，用户搜 AI 工具时很容易看到。

**需盘古**：注册知乎账号，开通专栏
