# 🚀 aibounty.cn 推广路线图

> 2026-07-01 | 曦和出品 | 零预算优先，每一分钱都花在刀刃上

---

## 核心策略：内容引擎 → 社区分发 → 增长飞轮

**不做：** 花钱投广告、刷量、买外链  
**只做：** 让开发者自己找到我们，来了就不想走

---

## Phase 0：地基加固（1-2天，零成本）

### 0.1 搜索引擎入场券 🎫

| 事项 | 现状 | 要做的 |
|------|------|--------|
| **sitemap.xml** | 只有4个URL，没有工具详情页 | **动态生成**所有1812个工具页URL |
| **robots.txt** | 基本ok | 补充各搜索引擎爬虫规则 |
| **meta标签** | 首页ok | 给每个分类/筛选状态加独立 meta 描述 |
| **结构化数据** | 无 | 加 JSON-LD（BreadcrumbList + WebSite） |
| **百度收录** | 未知 | 手动提交到百度搜索资源平台 |
| **Google Search Console** | 未配置 | 接入验证 |

**预期效果：** 2-4周后搜索引擎开始收录工具详情页，长尾流量来源

### 0.2 日报 → 持续内容输出 📰

每日AI工具日报已经有流程了，但还需要：

- **日报自动化发布**到公众号/知乎/掘金（先做1个平台跑通）
- **日报SEO化**：标题含长尾关键词、正文加内链
- **周报/月报**：对日报内容做聚合精华版

### 0.3 关于页 → 信任背书

about.html 现在太简陋，补充：
- 站点数据展示（实时：1812工具，9大来源）
- 更新频率说明（每日自动狩猎）
- 团队故事（曦和 + 盘古）
- 数据来源可信度说明

## Phase 1：社区渗透（3-7天，零成本）

### 1.1 GitHub 开源 → 开发者的阵地 🐙

在 GitHub 建一个 `aibounty` 仓库，放：

| 资产 | 作用 |
|------|------|
| `README.md` | 项目介绍 + 功能截图 + 站点链接 |
| 数据样例 | 展示data.json结构，吸引数据消费者 |
| 日报存档 | 历史日报markdown版 |
| 狩猎脚本（精简版） | 吸引开发者fork/star |

**分发渠道：**
- HackerNews Show HN（准备好英文版README）
- 掘金「站长工具 / 开源推荐」分类
- V2EX「分享创造」节点
- 开源中国(oschina.net)
- 阮一峰《科技爱好者周刊》投稿

### 1.2 技术社区 → 内容分发 🧑‍💻

**掘金：** 注册专栏《AI工具周刊》，每周发一篇
- 选题：《本周最值得关注的10款AI工具》
- 每个工具一句点评 + 链接回 aibounty.cn
- 底部自然露出「更多AI工具尽在 aibounty.cn」

**知乎：** 回答AI相关问题时顺带推荐
- 《有哪些好用的AI编程工具？》
- 《2026年最值得关注的AI开源项目》
- 回答结尾引导：aibounty.cn 每日狩猎更新

### 1.3 社交媒体 → 泛人群覆盖 📱

**小红书：** AI工具推荐笔记
- 标题：《每天知道一款AI工具 Day N》
- 图文卡片 + 引导搜索"aibounty"
- 笔记中不放外链（小红书限流），引导搜索

**抖音/视频号：** AI工具测评短视频
- 如果盘古有精力做短视频的话
- 没精力就PASS，不勉强

## Phase 2：增长飞轮（1-2周，低成本）

### 2.1 邮件订阅 💌

最简单的增长工具，用户只需要留邮箱就能收到每日精选。

- 接入方式：腾讯文档表单 / 钉钉邮件 / SendGrid 免费层
- 内容：每日AI工具精选5款 + 一句话点评
- 页面底部加订阅入口

### 2.2 CPS 联盟推广 💰

**思路：** AI工具很多有推广返佣，把我们的推荐流量变现

| 联盟 | 适合什么工具 | 佣金 |
|------|-------------|------|
| OpenAI 推荐 | ChatGPT Plus | 约$5/单 |
| 各种SaaS联盟 | Cursor/JetBrains/Notion | 10-20% 首单 |
| Affiliate 聚合 | Impact/ShareASale | 看具体产品 |

**做法：**
- 在data.json每个item加 `affiliate_url` 可选字段
- 卡片上显示「🛒 购买」按钮（带aff参数）
- 赞助位已有了，可以做更精准的推荐

### 2.3 SEO 精细化 📈

- **分类页URL化**：`aibounty.cn/category/agent` 等
- **标签页URL化**：`aibounty.cn/tag/price-open-source`
- 每个分类页加独立title/description
- 内链建设：每个工具详情页相关推荐 + 标签云

### 2.4 外链交换 🤝

联系同类AI导航站、AI博客、个人开发者：
- 在对方站加我们的友情链接
- 在对方的AI工具推荐帖下留言
- 免费提供"本网站AI工具由aibounty.cn提供数据"

## Phase 3：盈利闭环（2-4周）

### 3.1 赞助升级已有机位

赞助位现在随机展示，可以做成更正式的：
- 首页顶部banner位（月付）
- 分类列表置顶（月付）
- 日报冠名（次/周）

### 3.2 CPS → 被动收入

结合Phase 2.2，跑通至少1个联盟渠道，验证转化率。

### 3.3 API 服务（高级）

如果数据被外部需要，可以提供：
- `GET /api/tools` 免费版（限制次数）
- `GET /api/tools` 付费版（不限+优先更新）
- 对量化交易/投资分析场景提供AI工具趋势数据
